Proyecto web Mean Stack Javascript

La finalidad de este proyecto era aprender el desarrollo de una web con puras tecnologias Javascript.

Client Name

  • Industry: Tech
  • Website: xxxx
La apliacion web esta pensada para llevar la gestion de un consultorio medico.
En el sistema puedes dar de alta Doctores, Hospitales, subir imagenes, buscar rapidamente informacion previamente agregada. iniciar sesion con la autentificación de google entre otras cosas.

Project Requirements

  • Crear apliación con el MEAN STACK Javascript.
  • Alta de doctores, Hospitales, subida de imagenes, api de google, JSON WEB TOKEN.
  • Hospedar apliacion (serverlees) con los servicios de Amazon Web Services.

Objetivo

Mi finalidad de esto era Aprender el MEAN STACK Javascript con el desarrollo de esta apliación WEB, el tiempo aproximado que me tomo aprender y desarrollar esto fue de 25 dias y estoy muy contento con los resultados pues al mismo tiempo tambien aprendi a montar desde cero la apliación en AWS esto sin usar un servidor (serverlees).

Desarrollo

Hablando de cosas tecnicas es complicado acostumbrarte a programar un back end con solo javascript en este caso usar node.js con express.js fue un poco raro al principio, esto por el echo de que vengo de asp.net donde para cualquier cosa tienes el "intellisense de microsoft".
El usar una base de datos no realacional como Mongo DB le agrega un extra de dificultad.

Algo interesante de la actividad es que al finalizarla hice el despliegue en un servicio de amazon web serivices llamado S3 donde no necesito un servidor para hospedar el sitio (serverlees)

Resultado

Al finalizar la actividad tengo ya mi sito funcional, en el puedes hacer distintas cosas antes ya mencionadas,
sin duda se pueden agregar mejoras y nuevos modulos, pero para la finalidad de la practica estoy mas que satisfecho.

Como podras observar en las imagenes mi back-end lo monte en una instancia de windows server 2016, Asi como la base de datos.
El front-end esta el servicio de S3 de AWS. OJO - Esto no es lo mas viable pero como dije antes era solo una prueba para ver como se comportaba.
En fin hasta aqui llego el proyecto, fue divertido, estresante y lo odie hasta cierto punto pero al final queda pura sastifacción. Happy coding!
